# Elite SMP Theme Redesign: Discord Modern + Weed + Mac Miller

## Color Palette

### Mac Miller Inspired
- **Swimming Pink**: `#FFB6C1` - Soft pink from Swimming album
- **Circles Blue**: `#87CEEB` - Light blue, dreamy aesthetic
- **Artistic Purple**: `#9B59B6` - Creative, nostalgic vibes

### Discord Modern
- **Background Dark**: `#2C2F33` - Discord dark gray
- **Card Background**: `#23272A` - Slightly darker cards
- **Accent Purple**: `#5865F2` - Discord blurple
- **Text Light**: `#DCDDDE` - Discord text color

### Cannabis/Weed Aesthetic
- **Cannabis Green**: `#9ACD32` - Vibrant lime green
- **Forest Green**: `#228B22` - Deep forest green
- **Sage Green**: `#87A96B` - Muted, chill green

### Combined Palette
- Primary: Cannabis Green (`#9ACD32`)
- Secondary: Discord Purple (`#5865F2`)
- Accent: Swimming Pink (`#FFB6C1`)
- Background: Discord Dark (`#2C2F33`)
- Card: `#23272A`
- Text: `#DCDDDE`
- Muted: `#72767D`

## Design Principles

1. **Dark Mode First** - Discord-inspired dark backgrounds
2. **Organic Accents** - Cannabis green for CTAs and highlights
3. **Nostalgic Touches** - Mac Miller pink/blue for artistic elements
4. **Keep Minecraft Text** - Pixelated typography stays
5. **Smooth Gradients** - Blend colors like album covers
6. **Chill Vibes** - Relaxed, artistic, community-focused

## Typography
- Headers: Minecraft pixelated font (keep existing)
- Body: Clean sans-serif (Discord-like)
- Accents: Subtle gradients on important text

## UI Elements
- Buttons: Cannabis green with hover effects
- Cards: Dark gray with subtle borders
- Navigation: Discord-style dark bar
- Accents: Pink/purple highlights for artistic flair
